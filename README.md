Portuguese Songs App

Purpose

The Portuguese Songs App is a personal language-learning project.

The goal is to help users learn Portuguese through music by displaying Portuguese lyrics together with line-by-line translations.

This project is also a learning journey in web development. It is intentionally being built one small step at a time using HTML, CSS, JavaScript, GitHub, and GitHub Pages.

Current Features

* Displays a song title and artist.
* Displays Portuguese lyrics paired with their translations.
* Automatically generates lyric cards using JavaScript.
* Optimized for reading on an iPhone.
* Hosted on GitHub Pages.

Planned Features

* Multiple songs.
* Song selector.
* Import songs from raw text.
* AI-assisted translation.
* Multiple translation languages.
* Embedded YouTube playback.
* Personal notes.
* Search and study tools.

Project Philosophy

* Learn Portuguese through songs.
* Portuguese always appears first.
* Translation appears immediately below each lyric.
* Build the project one small step at a time.
* Keep the design simple and mobile-friendly.
